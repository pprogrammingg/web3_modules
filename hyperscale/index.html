Seven levels — three modules each (Level 6 adds a fourth Hyperscale-rs reading module on Rust idioms alongside measurement, E2E, and lab)—structured path toward elite decentralized protocol engineering (Levels 2–7 include one purple OSS contribution module; Level 1 is foundations + transaction-flow map only)

Who this trains

This is deliberate coursework for engineers growing along the whole ladder—junior, intermediate, senior, tech lead, staff, and beyond. The primary outcome is not “know Hyperscale-rs only”; it is to think like a protocol engineer: precise about consensus and economics, rigorous in Rust, calm under partial synchrony, and fluent in networking and ops when shipping nodes.

Hyperscale-rs is the cutting-edge lab—a serious Rust workspace where BFT, execution, cryptography, simulation, and libp2p meet. Every module ends with a Protocol engineer track panel: transferable Rust idioms, curated further reading, and protocol trivia so you round out as a professional, not only a tourist in one repo.

Course Structure

Each Level (1–7) lists three primary modules (Level 6 also includes an extra Rust optimizations Hyperscale-rs reading module between performance and E2E). Levels 2–7 pair two learning tracks with one OSS contribution module (purple card, tied to a hyperscale-rs issue). Level 1 is vocabulary → consensus → single-shard transaction flow so you see concrete Hyperscale wiring before long reading runs; your first hands-on OSS PR starts in Level 2. Contribution modules spell out steps and expected results so PR work feels guided.

  • Levels 1–3 — Foundations → systems thinking → Hyperscale orientation & crate map
  • Levels 4–5 — Sharding, BFT, cross-shard, then execution / timing / cryptography
  • Levels 6–7 — Measurement, E2E tests, lab project → libp2p depth & packaging

Hyperscale-rs–specific modules keep the Hyperscale-rs badge; contribution slots also show an OSS contribution pill with a light purple card treatment.

How to Use This Course

  1. Start at Level 1 - Even if you have some experience, the foundation modules anchor the rest of the path
  2. Progress by level number — Complete the modules in a level in order; Level 6 has four Hyperscale-facing steps (adds Rust optimizations between performance and E2E).
  3. Complete All Sections - Read content, take quizzes, complete assignments
  4. Focus on Hyperscale-rs Modules - If your goal is to contribute to hyperscale-rs, prioritize those modules
  5. Take Your Time - Each module is designed for 1.5-2 hours. Don't rush!
  6. Scroll to the Protocol engineer track — At the bottom of each module, curated Rust depth, readings, and protocol snacks scale with your level; revisit after promotions or role changes.
