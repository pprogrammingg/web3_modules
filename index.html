Course Structure

The index uses a single Level number from 1 through 11 (no separate “Basic” band name on the page):

  • Level 1 — Foundations (blockchain, consensus, distributed systems, state machines, Hyperscale orientation)
  • Levels 2–6 — Protocol deep dive, implementation, testing, simulation, storage, and intermediate contributions
  • Levels 7–11 — Advanced view changes, performance, security, scale, production, testing, and major contributions

Each level block uses a repeating color cycle so the path is easy to scan. Modules are marked with a Hyperscale-rs badge when they're specific to the hyperscale-rs codebase. Coming Soon modules still appear in their level so you can see the full roadmap.

How to Use This Course

  1. Start at Level 1 - Even if you have some experience, the foundation modules anchor the rest of the path
  2. Progress by level number - Follow Levels 2→6, then 7→11 when you can; skip only when you already know the topic
  3. Complete All Sections - Read content, take quizzes, complete assignments
  4. Focus on Hyperscale-rs Modules - If your goal is to contribute to hyperscale-rs, prioritize those modules
  5. Take Your Time - Each module is designed for 1.5-2 hours. Don't rush!
